What is this?
This is the default quantity threshold for low-stock alerts. When a product's stock falls below this number, you'll receive a notification.
Example: If set to 5, you'll be alerted when any product has 5 or fewer units remaining.
You can also set a custom threshold for individual products in the product edit screen.
